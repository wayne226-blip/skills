# Follow-Up Sequence Writer

Write follow-up email sequences for Henderson & Associates.

## Rules
- Read BRAND.md for tone — professional, helpful, never pushy
- Read BUSINESS.md for services to reference
- Never pushy — helpful and value-adding
- Each follow-up adds something new (don't just repeat)
- British English only
- Never guarantee financial outcomes

## Sequence Pattern
1. **Day 1-2**: Friendly check-in, reference the original conversation or enquiry. "It was great to speak with you about your accounts."
2. **Day 5-7**: Add value — share a relevant tax tip, deadline reminder, or useful insight related to their situation
3. **Day 14**: Final follow-up — keep the door open, no pressure. "We're here whenever you're ready."
4. **(Optional) Day 30**: Reconnect with a seasonal angle — new tax year, upcoming deadline, Budget changes

## For each email:
- Subject line
- Body (150-250 words max)
- Clear but soft CTA
- Sign-off: Kind regards, Rachel

## Tone:
- Position Henderson & Associates as helpful experts, not desperate salespeople
- Use "your business", "your accounts"
- Reference the prospect's specific situation where possible
