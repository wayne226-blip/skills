# Newsletter Writer

Write the monthly newsletter for Henderson & Associates.

## Rules
- Read BRAND.md for tone — authoritative but friendly
- Read BUSINESS.md for services and expertise areas
- Scannable — use headers, short paragraphs, bullet points
- One clear CTA per newsletter
- Subject line + preview text included
- British English only
- Never guarantee financial outcomes
- Be seasonally aware — reference relevant upcoming HMRC deadlines

## Structure
1. **Subject line** + **preview text** (under 90 chars for subject, under 140 for preview)
2. **Opening** — warm, timely hook (seasonal reference, recent Budget changes, upcoming deadline)
3. **Main article** — tax tip, regulatory change, or business insight (300-400 words, plain English)
4. **Deadline reminder** — what's coming up in the next 1-2 months
5. **Quick tip** — one actionable thing the reader can do this month
6. **CTA** — book a review, refer a friend, get in touch
7. **Sign-off** — personal, warm: Kind regards, Rachel

## Tone guidance:
- This is a client and prospect newsletter — it should position Henderson & Associates as experts while being genuinely helpful
- Avoid jargon — explain technical terms if used
- Use "your accounts", "your business", "we'll take care of it"
- Aim for "useful friend who knows about tax" not "accountant lecturing you"
- Brand colours for any formatting: Navy #003366, Gold #C4A35A, White #FFFFFF
