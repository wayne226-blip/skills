# Social Media Post Writer

Write social media posts for Bright Coaching (Sarah Chen).

## Before Writing

- Read BRAND.md for tone, visual identity (orange #FF6B35, navy #1A1A2E), and banned phrases
- Read BUSINESS.md for expertise areas and audience

## Rules

- Sarah's voice: direct, warm, not preachy. Like a straight-talking friend.
- Adapt format to platform
- British English only
- No banned phrases from BRAND.md
- Brand colours are #FF6B35 (orange) and #1A1A2E (dark navy) — reference these for any visual suggestions

## Per Platform

### LinkedIn (Primary platform)
- Professional thought leadership for mid-level tech managers
- 150-300 words
- Use line breaks for readability
- 3-5 hashtags max, specific: #leadershipcoaching #techleadership #managertips
- Strong hook in first line (shows before "see more")
- Real coaching observations (anonymised), practical tips, honest reflections
- See the LinkedIn Post Writer skill for detailed post structure

### Instagram
- More visual, personal, behind-the-scenes
- Shorter captions (50-150 words)
- 10-15 hashtags in a separate block
- Suggest image type: quote graphic (use brand colours), photo idea, carousel concept
- Coaching tips formatted as quick lists work well
- More casual than LinkedIn but still Sarah's voice

### Facebook
- Community-focused, conversational
- Encourage comments and discussion
- Shorter than LinkedIn
- 2-3 hashtags max
- Good for sharing wins, asking questions, community building

### X/Twitter
- Punchy, under 280 chars
- Thread format for longer ideas (max 5 tweets)
- No hashtags in body, 1-2 on final tweet
- Hot takes on leadership and management culture

```

Visual suggestion: [brief description of image/graphic if relevant]
```
