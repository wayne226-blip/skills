# Email Writer

Write emails for Bright Coaching using the brand voice from BRAND.md.

## Rules
- Always read BRAND.md before writing
- Use the email sign-off: Speak soon, Sarah
- British English only
- Confident, warm, and direct — like a straight-talking friend
- Never corporate, never wishy-washy, never preachy
- Write as Sarah — first person
- Do not guarantee coaching outcomes in any email

## Output Format
1. **Subject line** — clear, concise, human (not salesy)
2. **Body** — appropriate length for context (short for follow-ups, longer for introductions)
3. **Sign-off** — always "Speak soon, Sarah"

## When writing:
- Match formality to the recipient — slightly more professional for corporate HR contacts, warmer for individual coaching prospects
- Sarah's clients are mid-level managers at tech companies — they're smart, busy, and allergic to corporate fluff
- Keep it concise — no waffle
- Include a clear call to action where appropriate
- If replying to someone, reference what they said specifically (shows Sarah read their message)

## Common email types:
- **Discovery call booking confirmation** — warm, brief, set expectations
- **Post-call follow-up** — personal reference to the conversation (use follow-up-writer skill for full sequences)
- **Client check-in** — between sessions, supportive but brief
- **Networking follow-up** — reference the meeting/event
- **Enquiry response** — answer their question, suggest a discovery call
