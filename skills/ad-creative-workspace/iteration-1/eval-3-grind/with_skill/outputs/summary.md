# Grind Ad Creative — Summary

## Brand: Grind (grfrnd.com)
London-based specialty coffee brand known for compostable Nespresso-compatible pods, strong sustainability positioning, and a minimalist fashion-editorial visual identity.

## Output Format
JSON array for Nano Banana Studio Bulk Import.

## Brand DNA Research
Web search and fetch were unavailable. Brand DNA was compiled from existing knowledge of Grind's brand identity, packaging, positioning, and competitive landscape. The Brand DNA document includes:
- Visual system (black + cream + muted pastels per blend)
- Photography direction (warm, matte, minimal, editorial)
- Product details (compostable pods in slim boxes, colour-coded blends)
- Competitive context (vs Pact, Blue Bottle, Nespresso)
- 67-word Image Generation Prompt Modifier prepended to every prompt

**Note:** The brand DNA should be verified against the live site. Colors, fonts, and specific product details may need adjustment.
